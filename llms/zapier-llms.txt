## Zapier Connectors

Publicly available, agent-callable connectors for the apps Zapier integrates with. Each connector is a single artifact that's simultaneously an agentskills.io skill and a bundle of MCP `Tool`-shaped TypeScript modules — install it as a skill, import it as an npm package, run it as a CLI, or serve it as a local MCP server. Unlike the hosted Zapier MCP server, connectors run in your own environment as readable code, authenticating through a Zapier-managed connection or your own third-party credentials. Experimental; APIs may change.

- Source: [github.com/zapier/connectors](https://github.com/zapier/connectors)
