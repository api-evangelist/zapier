Zapier connects to thousands of apps, but maybe an app or action you need isn't available, or a native integration doesn't let you set OAuth scopes. Without them, IT might not approve the integration you want. In the past, there was just one workaround: to paste the API key for your desired app into Webhooks by Zapier steps.
